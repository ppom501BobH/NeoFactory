# 智慧工廠管理系統 - 啟動指南

## 🚀 快速啟動（3 步驟）

### 步驟 1：安裝依賴

```bash
cd smart-factory
npm install
```

### 步驟 2：啟動開發伺服器

```bash
npm run dev
```

### 步驟 3：開啟瀏覽器

系統會自動在瀏覽器開啟，或手動訪問：
```
http://localhost:5173
```

如果 5173 port 被佔用，系統會自動使用下一個可用 port（如 5174）。

## 📊 查看專案簡報

### 方法 1：直接開啟 HTML 檔案

在檔案總管中，雙擊開啟：
```
smart-factory/智慧工廠管理系統簡報.html
```

### 方法 2：使用瀏覽器開啟

在瀏覽器中按 `Ctrl + O`，選擇簡報檔案開啟。

### 簡報操作說明

- **鍵盤導航**：
  - `→` 或 `空白鍵`：下一頁
  - `←`：上一頁
  - `Home`：第一頁
  - `End`：最後一頁

- **觸控導航**：左右滑動切換投影片

- **按鈕導航**：使用畫面下方的控制按鈕

## 📸 生成系統截圖（選用）

簡報目前使用佔位符圖片。如需生成實際系統截圖：

### 步驟 1：確保開發伺服器運行中

```bash
npm run dev
```

### 步驟 2：在新終端執行截圖腳本

```bash
node screenshot-script.js
```

截圖會自動保存到 `screenshots/` 目錄，並替換簡報中的佔位符圖片。

**注意**：截圖功能需要 Playwright 瀏覽器支援。如果遇到問題，可以手動截圖並放入 `screenshots/` 目錄。

## 🛠️ 常見問題

### Q1: Port 被佔用怎麼辦？

系統會自動尋找可用 port。如需指定 port，修改 `vite.config.ts`：

```typescript
export default defineConfig({
  server: {
    port: 3000  // 改為您想要的 port
  }
})
```

### Q2: 如何建構生產版本？

```bash
npm run build
```

建構完成後，檔案會輸出到 `dist/` 目錄。

### Q3: 如何預覽生產版本？

```bash
npm run preview
```

### Q4: 截圖腳本執行失敗？

可以手動截圖：
1. 開啟系統各個頁面
2. 使用 `Win + Shift + S` 截圖
3. 將截圖保存為 PNG 格式到 `screenshots/` 目錄
4. 檔名：dashboard.png, production.png, equipment.png, quality.png, traceability.png, issues.png
