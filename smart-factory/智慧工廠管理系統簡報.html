生產管理功能

📋
工單管理
20+ 筆工單資料，優先級標示，狀態追蹤，進度條視覺化

統計資訊

  • 開始/結束時間、所屬產線
  • 優先級管理（高/中/低）
07

設備管理功能

設備監控與維護

  • 設備列表 - 20+ 台設備即時監控，運轉時數統計
  • 參數監控 - 溫度、振動、轉速、電流即時顯示
  • 設備告警 - 即時告警通知，嚴重程度分級
  • OEE 分析 - 可用率、表現率、品質率綜合分析
  • 保養計畫 - 定期保養排程，逾期提醒

追溯管理功能

完整批次履歷

  • 原料追溯 - 原料使用記錄，批次號追蹤
  • 品質記錄 - 檢驗時間與結果，不良類型記錄
  • 操作員資訊 - 作業時間統計，責任追蹤

# 2. 安裝依賴

npm run dev
