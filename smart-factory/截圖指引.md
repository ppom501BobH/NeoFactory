# 智慧工廠管理系統 - 截圖指引

- 建議尺寸：1920x1080

### 3. 設備管理 (equipment.png)
- 路徑：http://localhost:5174/equipment

### 4. 品質管理 (quality.png)

### 5. 追溯管理 (traceability.png)

## 截圖方法

### 方法 1：手動截圖
1. 開啟瀏覽器到指定頁面
2. 按 F11 進入全螢幕模式
3. 使用 Windows 截圖工具（Win + Shift + S）
4. 將截圖保存到 `screenshots/` 資料夾，使用上述檔名

### 方法 2：使用自動截圖腳本
執行以下命令：
```bash
node screenshot-script.js
```

## 截圖保存位置

所有截圖應保存在：
```
smart-factory/screenshots/
├── dashboard.png
├── production.png
├── equipment.png
├── quality.png
├── traceability.png
└── issues.png
```

截圖完成後，簡報會自動載入這些圖片。
