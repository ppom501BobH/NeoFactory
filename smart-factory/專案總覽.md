## 🚀 如何啟動系統

### 方法 1：快速啟動（推薦）

```bash
cd smart-factory
npm install
npm run dev
```

然後在瀏覽器開啟：`http://localhost:5173`

### 方法 2：查看詳細指引

開啟 [`啟動指南.md`](./啟動指南.md) 查看完整說明。

## 📊 如何查看簡報

### 直接開啟 HTML 檔案

在檔案總管中，雙擊開啟：
```
智慧工廠管理系統簡報.html
```

### 簡報操作

- `→` 或 `空白鍵`：下一頁
- `←`：上一頁
- `Home`：第一頁
- `End`：最後一頁

## 📸 系統截圖

簡報目前使用 SVG 佔位符圖片。如需生成實際系統截圖：

```bash
# 1. 確保系統運行中
npm run dev

# 2. 在新終端執行
node screenshot-script.js
```

截圖會自動保存到 `screenshots/` 目錄並顯示在簡報中。
