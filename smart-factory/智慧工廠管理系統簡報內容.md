## 投影片 2：專案概述

### 系統簡介
- 基於 React + TypeScript 的現代化 Web 應用
- 提供完整的工廠生產管理與監控功能
- 即時數據視覺化與分析
- 響應式設計，支援多種裝置

### 核心價值
- 提升生產效率
- 降低管理成本
- 即時問題追蹤
- 數據驅動決策

---

## 投影片 3：技術架構

### 前端技術棧
- **框架**: React 19.2.6 + TypeScript 6.0.2
- **建構工具**: Vite 8.0.12
- **UI 框架**: Ant Design 6.4.4
- **圖表庫**: ECharts 6.1.0
- **狀態管理**: Zustand 5.0.14
- **路由**: React Router 7.17.0
- **樣式**: Tailwind CSS 4.3.1

### 架構特點
- 單頁應用 (SPA)
- 組件化開發
- 模組化設計
- 零依賴部署

## 投影片 16：部署與啟動

### 快速開始
```bash
# 1. 進入專案目錄
cd smart-factory

# 2. 安裝依賴
npm install

# 3. 啟動開發伺服器
npm run dev

# 4. 開啟瀏覽器
http://localhost:5174
```

### 建構生產版本
```bash
npm run build
```
